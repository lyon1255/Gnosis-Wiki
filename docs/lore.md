# Main Lore: The Sunken Gates

## The World
Aethelgard was consumed by the **Aether Storm**, a magical anomaly that trapped the survivors in **Brink Valley**. The only way out is down into the **Sunken Babel**, a series of simulated dimensions.

## Core Systems
| System | Description |
| :--- | :--- |
| **The Breaking** | A global event where a Boss is defeated for the first time, opening the gate for everyone. |
| **The Echo** | A weakened shadow of a defeated Boss, allowing solo players to finish their Main Quests. |
| **Ley-Lines** | The teleportation network connecting the Brink Valley to the Gates. |

## Faction Rules
- **Faction 0 (Player):** Reserved ONLY for actual players.
- **Faction 1 (Neutral):** Townsfolk, Vendors, Quest Givers.
- **Faction 2 (Enemy):** Monsters, Bosses, Hostile NPCs.
