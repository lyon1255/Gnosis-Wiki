# Reference: Items & Rarity

## Item Categories
| Category | Description |
| :--- | :--- |
| **CONSUMABLE** | Potions, food, temporary buffs. |
| **EQUIPMENT** | Weapons, armor, accessories. |
| **MATERIAL** | Crafting ingredients (Ores, Gems). |
| **TRASH** | Items intended only for selling to vendors. |
